# serverMC_termux
Инструмент для создания сервера Java Minecraft в Termux с открытым исходным кодом.

 

Установка:

    apt update && apt upgrade -y
    apt install -y git
    git clone https://github.com/erophey7/serverMC_termux.git
    cd serverMC-termux
    chmod +x install.sh
    ./install.sh
